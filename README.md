# ⏱️ Stopwatch Web App

A modern and responsive **Stopwatch Web Application** built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. The application allows users to accurately measure time, pause/resume, reset the timer, and record lap times with a clean and intuitive interface.

---

## ⚙️ Functionality

### ▶️ Start
- Starts the stopwatch.
- Prevents multiple intervals from running.

### ⏸️ Pause
- Stops the timer while preserving the current time.

### 🔄 Reset
- Stops the timer.
- Resets minutes, seconds, and milliseconds.
- Clears all lap records.

### 🏁 Lap
- Saves the current stopwatch time.
- Displays the latest lap at the top.
